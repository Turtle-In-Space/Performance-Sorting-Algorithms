NAME
==================================================

**benchmark.sh** - Benchmark sorting algorithms and plot them against eachother

SYNOPSIS
==================================================

**benchmark.sh** COMMAND

DESCRIPTION
==================================================

Benchmark sorting algorithms and plot them against eachother


COMMANDS
==================================================

benchmark.sh build
--------------------------------------------------

Compiles and builds all scripts

benchmark.sh clean
--------------------------------------------------

Deletes all build- and program made files

benchmark.sh run
--------------------------------------------------

Runs all sorting algorithms

benchmark.sh plot
--------------------------------------------------

Plots all data collected with run command

benchmark.sh completions
--------------------------------------------------

Generate bash completions
