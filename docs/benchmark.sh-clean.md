NAME
==================================================

**benchmark.sh clean** - Deletes all build- and program made files

SYNOPSIS
==================================================

**benchmark.sh clean** [OPTIONS]

DESCRIPTION
==================================================

Deletes all build- and program made files


OPTIONS
==================================================

--lang LANG
--------------------------------------------------

Name of coding languaes [LANG1,LANG2,...]


--algo ALGO
--------------------------------------------------

Name of sorting algorithms [SORT1,SORT2,...]


EXAMPLES
==================================================

~~~
benchmark clean

benchmark clean --lang java

~~~
